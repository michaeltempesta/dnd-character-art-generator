# 🎨 D&D Character Art Generator - Unified Edition

**One-click AI art generation for your D&D characters**

Transform your D&D Beyond character sheets into stunning artwork using state-of-the-art AI models. **No coding required** - just upload to Colab and start generating!

## ✨ Features

### 🎭 **5 Art Styles**
- **Fantasy Realistic**: Photorealistic character portraits
- **Epic Fantasy**: Heroic, dramatic character art  
- **Dark Fantasy**: Gothic, mysterious atmosphere
- **Anime Style**: Japanese animation aesthetic
- **Watercolor Fantasy**: Hand-painted artistic look

### 🤖 **AI-Powered Enhancement**
- **OpenAI Integration**: Intelligent prompt optimization
- **Character Analysis**: Extract visual characteristics from character data
- **System Diagnostics**: AI-powered performance optimization
- **Auto-Configuration**: Intelligent defaults based on your system

### 🛠️ **Advanced Tools**
- **SDXL Technology**: Latest Stable Diffusion XL models
- **ControlNet Integration**: Line-Art and Tile ControlNet for precise control
- **Refiner Support**: Optional SDXL Refiner for enhanced quality
- **Inpaint Capabilities**: Surgical fixes and detailed editing
- **Mask Helper**: Integrated mask creation tools
- **ComfyUI Export**: Export workflows to ComfyUI for advanced users

### 🔒 **Security & Performance**
- **Hardened Security**: Auth, input validation, content filtering
- **Auto-Optimization**: System-specific performance tuning
- **Memory Management**: Intelligent VRAM usage
- **Real-time Monitoring**: Performance diagnostics and optimization

## 🚀 Quick Start

### For Complete Beginners
1. **Upload `main_notebook.ipynb` to Google Colab**
2. **Click "Runtime" → "Run all"**
3. **Wait for setup (2-3 minutes)**
4. **Click the generated link**
5. **Start creating art!**

### For Advanced Users
1. **Configure API keys in Cell 3** (optional but recommended)
2. **Customize settings in the app**
3. **Use advanced tools in Cell 5**
4. **Export to ComfyUI for complex workflows**

### For Local Development
```bash
git clone <repository-url>
cd dnd-character-art-generator
pip install -r requirements.txt
python -m apps.unified_app
```
